# TP4 : Intégration Web

> Remise : 17 juillet\
> Modalité : en équipe de 2

Tout au long de la session, vous travaillerez en équipe de deux à la
refonte d'un site Web de votre choix. Chaque travail pratique (TP)
correspondra à l'un des éléments livrables du processus de conception
d'une interface Web : charte graphique, maquette conceptuelle, maquette
graphique et intégration.

Pour ce quatrième et dernier TP, votre tâche consiste à intégrer les
maquettes graphiques sur lesquelles vous avez travaillé pour le TP3.

## Consignes

- L'intégration des maquettes doit se faire à l'aide du
  [générateur de sites statiques] vu en classe.
- Les feuilles de style doivent respecter la méthodologie BEM.

[générateur de sites statiques]: https://github.com/582-21W-MA/ssg

## Remise

Pour que votre travail soit corrigé, le code source doit être téléversé
dans le dépôt Classroom créé pour le TP1, _et_ le site statique doit
être hébergé sur votre serveur du Collège.

## Présentation

Lors de la remise, vous devrez présenter votre travail à la classe (et
aux parties prenantes imaginaires). Votre présentation ne doit pas
dépasser 10 minutes, et votre caméra doit être ouverte durant celle-ci.
Pendant la présentation :

- présentez la version _desktop_ et _mobile_ des pages Web ;
- expliquez les ajustements faits aux maquettes graphiques (s'il y en a)
  ; et
- identifiez quelques points à améliorer pour votre prochain projet.

## Critères d'évaluation

- Intégration [7 points]
  - Le balisage est [valide][w3c] et sémantique.
  - Le site s'adapte bien aux différentes tailles d'écran.
  - Les conventions de nommage BEM sont respectées.
  - Les maquettes graphiques sont intégrées fidèlement.
  - Les gabarits contiennent peu ou pas de logique.
  - Les parties de l'interface qui se répètent sont abstraites dans des
    gabarits partiels.
  - Le code est formaté de manière cohérente.

- Présentation [3 points]
  - Les ajustements apportés aux maquettes sont bien expliqués.
  - La présentation est soignée et bien structurée.

Outre ces critères, vous pouvez être pénalisé·es si votre travail ne
respecte pas les consignes, ou si la charge de travail n'est pas divisée
également.

[w3c]: https://validator.w3.org/#validate_by_input
